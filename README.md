# 🧮 Calculator App

A simple intermediate-level calculator built using HTML, CSS, and JavaScript.

---

## 🚀 Features

- Basic operations (+, −, ×, ÷)
- Keyboard support ⌨️
- Calculation history 📜
- Clear & delete options
- Error handling
